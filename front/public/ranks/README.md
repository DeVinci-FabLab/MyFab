# Icônes de rang (classement)

Images de rang utilisées par `/panel/classement`. Un fichier **par sous-tier**.
Si un fichier manque, la page retombe sur un losange coloré généré (pas d'erreur).

## Noms de fichiers (PNG)

`radiant.png`, puis `<tier>-<niveau>.png` :

- `immortal-3.png` `immortal-2.png` `immortal-1.png`
- `ascendant-3.png` `ascendant-2.png` `ascendant-1.png`
- `diamond-3.png` `diamond-2.png` `diamond-1.png`
- `platinum-3.png` `platinum-2.png` `platinum-1.png`
- `gold-3.png` `gold-2.png` `gold-1.png`
- `silver-3.png` `silver-2.png` `silver-1.png`
